Variable evt

Forma de capturar el evento de un elemento con JavaScript: 
document.addEventListener("click",evt=>{
  
});
La variable evt al ser mostrada en la consola tiene varias propiedades llamadas:
  1. target

    El "punto" que recibe el ratón (el click es un evento de "click"), por tanto hace referencia al elemento objeto sobre el cuál ha se ha asignado el evento.

  2. currentTarget

    El elemento del que parte el evento, siempre se refiere al elemento del event Handler (Manejador de Eventos) que ha sido atachado (adjuntado) a un event.target especifico, este identifica el elemento en el que se produjo el evento.

  3. relatedTarget

    Read-only property is the secondary target for the mouse event, if there is one

Con el título de esta leyenda ya podemos intuir que existen dos formas de generar funciones en JavaScript.

Definición

Una función es un procedimiento en JavaScript, es decir, un conjunto de sentencias que realizan una tarea o calculan un valor. Para usar una función, debe definirla en algún lugar del ámbito desde el cual desea llamarla.

Sintaxis

function miFuncion(parametro1, parametro2){//bloque de sentecias;[return valor;]}

Nueva Sintaxis

A partir de J6 en adelante surge una nueva sintaxis para definir las funciones anónimas que retornan un valor. Ahora se utilizar el operador flecha o arrow (=>)

Sintaxis

let varible=(parametro1, parametro2)=>{//bloque de sentecias; return valor;}
